# Mock Testing Suite FAQ

## What if the candidate stops responding?
Use the red Stopped Responding button to end the session as a fail.

## What if the candidate has technical issues?
Use Tech Issue and follow the troubleshooting flow for internet, routing, browser, or other issues.

## Can I go back and change something?
Yes. Use Back on the session screens. The active session is saved as you go.

## How do I run a Supervisor Transfer Only session?
From Home, choose Supervisor Transfer Only and either resume eligible prior mock calls or start a fresh supervisor-only session.

## Where is my data stored?
Session and settings data are stored in the local app database.

## How do I customize Discord templates?
Open Settings and use the Discord tab to edit templates and screenshots.
